How to read it

Wikipedia includes articles about 2,424,305 humans. Surprisingly, it also has articles about 588,822 different Chinese villages.

Technical note

The type of each item on Wikipedia is defined by the property “instance of” (P31)
on Wikidata. On July 28th 2014, we collected 12,607
different types using Magnus Manske’s WikiDataQuery
API. We have selected
the 781 types describing at least 100 items. These 781 types cover 99,16% of the whole dataset. We have arranged them into 9 main categories (living organism, wikimedia, geographical entity, creative work, group of humans, physical entity, symbolic entity, manufactured object, and temporal entity).

The categories tree is displayed using the Reingold-Tilford algorithm.
