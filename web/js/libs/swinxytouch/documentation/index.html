C) Package Contents - top

This package contains all sources and a combined minified version of all scripts, for quick deployment out of the box the recommended files are :-

libs/jquery-1.8.3.min.js
jquery.swinxy-combined.min.js

jquery.swinxy-combined.min.js :-
Combined and minified JS file encompassing the below files, if you wish to make the file smaller you may repackage with just the files you need but be sure touch.js is always the first within your new minified file.

  • sources/jquery.swinxytouch/touch.js
  • sources/jquery.swinxytouch/gesture-doubletap.js
  • sources/jquery.swinxytouch/gesture-drag.js
  • sources/jquery.swinxytouch/gesture-longpress.js
  • sources/jquery.swinxytouch/gesture-rotate.js
  • sources/jquery.swinxytouch/gesture-scale.js
  • sources/jquery.swinxytouch/gesture-swipe.js
  • sources/jquery.swinxytouch/gesture-tap.js
  • sources/jquery.swinxytouch/helper-smartclick.js
  • sources/jquery.swinxytouch/helper-bound.js
